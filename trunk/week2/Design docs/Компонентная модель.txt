Вся система построенная за принципом модели MVC. За базу данных взята Oracle Database Express Edition 11g Release 2. 
Взаимодействие с базой данных происходит с помощью библиотеки Hibernate.  В качестве веб-сервера был использован 
Apache TomCat версии 7.0.27. За отображение информации (визуализацию) отвечает html, css и javascript. 
Отдельные элементи, типа treeView и таблицы рисовались с помощью jquery. 
Для всего это использовался в том числе фреймворк bootstrap. Поддержка интернационализации и 
последующей многоязычности была осуществлена с помощью бибилиотеки jstl.
Использовались javaBeans для повторно используемых програмных компонентов, а 
так же сервлеты для взаимодействия с клиентами.
